# Next ORM Microsoft SQL Server provider

Nextorm - high performance zero-sql object-relational mapping (ORM) library

## Installation

- from cli  
```dotnet add package nextorm.sqlserver```
- from package manager  
```Install-Package nextorm.sqlserver```
